Clustering

In this project we will cover the concept of clustering, which is a unsupervised learning algorithm that involves grouping similar data points together based on their characteristics. The goal of clustering is to find similarities within a dataset and group similar data points together while keeping dissimilar data points separate.

Think of this project from a business perspective. Based on the customer profile, the business can identify different clusters and customize the experience, offers, services, products, and others based on this clusterization.

To check the analysis on GitHub, click here. Below, I explain a little bit about the Data analytics methodology, tasks involved in the project, data and code source, and the conclusions.

Data source

For this project we will be using a dataset called "Credit Profile" available at kaggle. Click here to access.

The dataset provides a comprehensive overview of potential loan applicants' profiles, specifically tailored for the Indian demographic. There are 14 features that are available:

Feature Type Description
Age Integer Represents the age of the applicant
Gender Categorical Gender of the applicant
Income Integer The applicant's income, which is critical in assessing their ability to repay the loan
Credit Score Integer A score quantifying the applicant's creditworthiness based on their credit history (300 to 850)
Credit History Length Integer Represents the number of months since the applicant's first credit line
Number of Existing Loans Integer The number of loans the applicant currently has
Loan Amount Integer The amount of money the applicant is requesting
Loan Tenure Integer The number of months the applicant wants to repay the loan over
Existing Customer Categorical Whether the applicant is an existing customer of the finance company
State Categorical The state in India where the applicant resides
City Categorical The city or village in India where the applicant resides
LTV Ratio Float The loan-to-value ratio, represents the ratio of the loan amount to the appraised value of the asset
Employment Profile Categorical General employment category of the applicant (Salaried, Self-Employed, Freelancer, Unemployed, Student)
Occupation Categorical Specific occupation or job title of the applicant
Profile Score Integer Represents the overall profile of the applicant based on the actual loan repayment data
